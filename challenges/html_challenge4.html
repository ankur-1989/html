There are three types of lists:

definition lists
which shows terms and definitions
unordered lists
which shows bullets
ordered lists
which shows numbers or letters

Above is an example of definition list.

Unordered list can have a type attribute
which can be applied to the UL or the LI tag.
The three values allowed are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Ordered lists can have a type attribute too.

Used along with a start attribute they can show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbers
